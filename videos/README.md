These files are optional local fallbacks.

The app now attempts to load open-source exercise media first (from the exercemus dataset).  
If the network/source is unavailable, it uses files from this folder.

Use these exact filenames:

- `pushup-good.mp4`
- `pushup-bad.mp4`
- `squat-good.mp4`
- `squat-bad.mp4`
- `pullup-good.mp4`
- `pullup-bad.mp4`
- `lunge-good.mp4`
- `lunge-bad.mp4`
- `situp-good.mp4`
- `situp-bad.mp4`
- `bicep_curl-good.mp4`
- `bicep_curl-bad.mp4`
- `shoulder_press-good.mp4`
- `shoulder_press-bad.mp4`
- `jumping_jack-good.mp4`
- `jumping_jack-bad.mp4`

Recommended export settings for mobile WebView:

- Resolution: `1280x720`
- Codec: `H.264`
- FPS: `24` or `30`
- Target bitrate: `2.5-4 Mbps`
- Audio: optional (muted by default in app)
